## Chat Application
### Global Dependencies
- npm
- serve
### Installation & Run
#### chat-example-server
```
cd chat-example-server
npm install
node index.js
```
#### chat-example-client
```
cd chat-example-client
serve -p 3001
```
#### chat-example-angular
```
cd chat-example-client-angular
bower install
serve -p 3002
```
